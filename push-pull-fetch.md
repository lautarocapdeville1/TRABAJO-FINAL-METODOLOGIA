# Git push
Sirve para subir tus commits locales al github. Con este comando mandas los commits que hiciste desde tu compu al repositorio para que los demas lo puedan ver.

# Git pull
Es el comando para traer los cambios que haya en github a tu compatudora. Lo usas cuando tus compañeros subieron cosas y queres tener el repo actualizado.

# Git fetch
Es parecido al git pull , solo que descarga los cambios , no los aplica. Te permite ver que hay nuevo antes de decidir si queres incorporarlo.
